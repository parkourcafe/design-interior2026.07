# Банк вопросов для брифа на дизайн интерьера

> Как собрано: автоматический веб-сбор (deep-research) нашёл релевантные источники,
> но не смог загрузить их содержимое (сетевые ограничения среды). Поэтому банк
> синтезирован из экспертной базы по интерьерным брифам и типовым чек-листам студий.
> Список найденных источников — в конце, для ручной проверки. Тон вопросов —
> поведенческий (как договаривались), валюта — рубль.

Легенда: каждый вопрос → **зачем** (какое проектное следствие). ✅ = уже есть в нашем брифе, ➕ = кандидат на добавление.

---

## 1. Объект / локация
- ✅ Тип объекта, площадь, город → масштаб проекта, региональные ставки.
- ✅ Район, этаж, тип дома (новостройка/вторичка/частный) → логистика, лифт/подъём, окна, инсоляция.
- ➕ **Состояние объекта сейчас**: пустая коробка / черновая / жилая с ремонтом → объём демонтажа, сроки, бюджет.
- ➕ **Планируется перепланировка?** двигать стены, мокрые зоны, объединять? → состав работ, согласования, сроки, цена. *(Один из главных вопросов — часто всплывает поздно.)*
- ➕ **Высота потолков, тип стен/перекрытий** → возможности по свету, нишам, потолкам.

## 2. Состав семьи и сценарии жизни
- ✅ Кто живёт сейчас / кто добавится за 5 лет (дети, родители, животные) → зонирование под будущее.
- ✅ Утренний сценарий, нагрузка на санузел → второй санузел / раздельный.
- ✅ Готовка (как часто, на скольких) → кухня, техника.
- ➕ **Работа из дома / хобби**: нужен кабинет, мастерская, спорт-зона? → отдельная зона, розетки, шумоизоляция.
- ➕ **Приём гостей**: как часто, сколько, ночуют ли → гостевое спальное место, обеденная зона.
- ➕ **Кто принимает решения** (один ЛПР или муж/жена/родители) → количество согласований и правок. *(Поведенческий прокси к рискам «затянется».)*

## 3. Бюджет
- ✅ Бюджетный коридор на ремонт и комплектацию → ценовой уровень, материалы.
- ➕ **Разделение бюджета: проект / ремонт-работы / мебель-комплектация** → снимает шок «а почему так дорого». *(Клиенты почти всегда путают эти три числа — сильнейший источник конфликта.)*
- ➕ **Что важнее в связке бюджет–срок–качество** (треугольник приоритетов) → куда двигаться при ограничениях.
- ➕ **Что из мебели/техники оставляем** (переезжаем со своим или всё новое) → планировка и комплектация.

## 4. Сроки
- ✅ Когда хотят закончить + флаг срочности → множитель срочности, реалистичность.
- ➕ **Жёсткий дедлайн и его причина** (свадьба, рождение, конец аренды, выход из ипотеки) → эмоциональный якорь + риск нереалистичных ожиданий по индивидуальной мебели.

## 5. Стиль и референсы
- ✅ 3–5 референсов + что точно НЕ нравится → визуальное направление, анти-паттерны.
- ➕ **Отношение к цвету и «смелости»** (нейтрально/ярко, готовы к акцентам?) → палитра.
- ➕ **Любимые/нелюбимые материалы и фактуры** (дерево, металл, глянец, камень) → отделка, экономические конфликты.

## 6. Функция и хранение
- ✅ Что сейчас не помещается → нагрузка на хранение.
- ➕ **Особые сценарии хранения**: гардеробная, кладовая, спорт-инвентарь, сезонное, коляска/велосипед → отдельные системы.
- ➕ **Бытовые привычки**: посудомойка/сушилка/робот-пылесос, стирка-сушка → инженерия и места.

## 7. Технические / строительные ограничения
- ➕ **Инженерия и комфорт**: тёплый пол, кондиционирование, вентиляция, умный дом, доп. розетки → бюджет и сроки, согласования.
- ➕ **Здоровье**: аллергии, астма (гипоаллергенные материалы), маломобильные члены семьи → материалы, безбарьерность.
- ➕ **Шум/приватность**: шумные соседи, спальня у лифта → шумоизоляция.
- ➕ **Ограничения дома**: несущие стены, газ, историческое здание, УК/согласования → что реально можно.

## 8. Коммерческие вопросы (что КЛИЕНТ спрашивает у дизайнера)
Их полезно предугадать и закрыть в КП:
- **Сколько стоит и из чего складывается цена?** → прозрачная разбивка (у нас есть в КП).
- **Что входит в дизайн-проект?** (планировки, визуализации, чертежи, ведомости, спецификации) → блок «что входит».
- **Сроки по этапам?** → этапы и сроки.
- **Сколько бесплатных правок?** → лимит правок.
- **Что такое авторский надзор и сколько стоит?** → пакет/опция.
- **Помогаете с закупкой/комплектацией?** → отдельная услуга / exclusions.
- **Договор, предоплата, гарантии — что если не понравится?** → условия.
- **Ведёте ли смету и контроль подрядчика?** → границы ответственности.
- **Примеры реализованных объектов (фото, не только рендеры)?** → доверие.

---

## «Чего обычно не спрашивают, но зря»
1. **Горизонт владения жильём** (для себя / продажа / аренда) — сильнейший фильтр бюджета и кастомизации. *(У нас ЕСТЬ — это наше преимущество.)*
2. **Разделение бюджета на 3 части** (проект/ремонт/мебель) — корень конфликта «дорого».
3. **Кто реально принимает решения** — предсказывает число правок и срыв сроков.
4. **Что делаем с текущей мебелью** — ломает планировку, если всплывает поздно.
5. **Жёсткий дедлайн и его причина** — эмоция, которую надо учесть в сроках.
6. **Здоровье/аллергии/животные** — влияет на материалы, а спрашивают редко.

## Сопоставление с нашим брифом (10 JTBD-вопросов)

**Уже покрыто:** объект (+район/этаж/тип дома), горизонт актива, домохозяйство, утро, готовка, хранение, бюджет, сроки, стиль, pain points, свободные комментарии к каждому вопросу.

**Рекомендую добавить (по убыванию ценности, 5–7 шт.):**
1. **Перепланировка** (готовы двигать стены / уже есть идеи) — choice: нет / думаем / да.
2. **Состояние объекта** — choice: пустая коробка / черновая / жилая с ремонтом.
3. **Бюджет: раздельно проект / ремонт / мебель** — или хотя бы уточнение «бюджет включает мебель?» (да/нет).
4. **Что из мебели оставляем** — multi: всё новое / часть оставляем / переезжаем со своим.
5. **Кто принимает решения** — choice: я один / вдвоём / вся семья (поведенческий прокси к правкам).
6. **Особые требования/здоровье** — multi: тёплый пол / кондиционер / умный дом / аллергии / маломобильность / нет.
7. **Жёсткий дедлайн** — необязательный текст: «есть дата, к которой надо въехать? почему?».

Каждый — с показом `show_if` и маппингом в паспорт + правило риска (напр. «перепланировка + срочность → timeline», «бюджет без мебели + премиум-референсы → budget»).
